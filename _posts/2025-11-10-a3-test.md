TFAR is a mod for Arma 3 that adds realistic positional VOIP as well as a highly configurable radio system. It is required to participate in Dog House Arma 3 operations.

## Requirements
- [ ] Teamspeak 3
- [ ] Task Force Radio Arrowhead

## Install Guide
1. Download and install the `TeamSpeak 3 client`: https://www.teamspeak.de/en/download/
2. Subscribe to the `Task Force Arrowhead Radio` mod from the Steam Workshop: https://steamcommunity.com/sharedfiles/filedetails/?id=894678801
3. Start Arma 3 and in the launcher select the `Task force Arrowhead Radio` mod -> display more options -> open folder in Windows Explorer.
![tar-more-options](assets/images/posts/a3-test/tfar-more-options.png)
4. Open the folders: `@Task Force ARrowhead Radio (BETA!!!)` -> `teamspeak`
5. Run the file `task_force_radio.ts3_plugin`

Task force radio is now installed. Yay!

## Keybinds

| Keybind     | Action                               |
| ----------- | ------------------------------------ |
| CAPS        | Short Range Radio Broadcast          |
| CTRL + CAPS | Long Range Radio Broadcast           |
| CTRL + P    | Short Range Radio Configuration Menu |
| ALT + P     | Long Range Radio Configuration Menu  |
| NUMPAD 1-9  | Cycle through preset Radio Channels  |
